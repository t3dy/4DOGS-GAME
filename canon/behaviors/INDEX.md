## How to add a new behavior

1. Create `canon/behaviors/B###_<dog>_<short>.md`. Use the structure of an existing entry (e.g., B005 or B012).
2. Fill all four era proposals (TEXT / ULTIMA5 / SIMS / NES). The NES constraint in particular exposes hidden complexity.
3. Add cross-references to related behaviors.
4. Add a row to this index.
5. If a behavior triggers a new system (schedules, broadcast events, household rules), note it under Design notes.
