## Writing conventions

- Each file uses the same cast.
- Dialogues have numbered **rounds** (not scenes) with topic headings.
- Each round typically ends with a concrete output: a decision, a list of questions, a compromise.
- Keep them readable as transcripts. No stage directions beyond a parenthetical or two.
- Aim for ~200-400 lines per dialogue. Longer and they start losing their shape.
