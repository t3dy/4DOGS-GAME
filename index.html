WHAT THIS IS

Built from real-world dog observations. Each behavior in the game corresponds to something one of these dogs actually does. The blueberry bush is dead in real life; Okie really did kill it.

Engine notes, character sheets, era-mockups, and design dialogues live in this repo under canon/ and design/. The game runs in your browser — no install, no server, state persists in localStorage between reloads.

Voice: Deja Vu / Police Quest noir.
Stack: vanilla HTML/CSS/JS, single file.
Companion: a Python terminal version under code/dogsim.py.
